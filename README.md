# Blurred Spoiler

##Usage

Include spoiler.js. Then, add this before &lt;/body&gt;:
```javascript
spoiler();
```

To control the maximum and partial blurs, you can pass arguments:
```javascript
spoiler({max: 10, part: 4});
```

Have fun! Play with it!
